## Technologies
Project is created with:
* React version: 17.0.2
* Redux version: 4.1.2
* Bootstrap version: 5.1.3
* Node version: 16.13.2
* Express version: 4.17.3
* Mongoose version: 6.2.8
* Passport version 0.5.2
